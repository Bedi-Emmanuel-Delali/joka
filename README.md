# Joka Studio

Open `index.html` in a browser to run the demo site.

If GitHub Pages is configured to publish from `docs/`, use `docs/index.html` as the published entry point.

GitHub Pages URL for this repository, once enabled, should be:
`https://Bedi-Emmanuel-Delali.github.io/joka/`

If that link does not load the interface, check the Pages source setting in GitHub and make sure it points to either:
- `main` branch + `/root`
- `main` branch + `/docs`

Features included:
- Client sign-up and sign-in
- Seeded admin access
- Live order tracking and status updates
- Cross-tab synchronization through browser storage and BroadcastChannel
